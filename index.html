About

The ZotBins System is a website with “smart bins” deployed in various locations (i.e. buildings, college campuses, cities) that can accurately measure and record the amount of waste in each bin.

The main purpose of this project is to help keep communties clean, make waste management more efficient, and encourage the practice of zero waste (diverting waste away from landfill).

Live Site

Features

Collect real-time data
- Data collection on bin fullness and weight of waste
- Calculates useful statistics such as divergence rates, or bin usage frequency
- Determines the effectiveness of zero waste resources (i.e. more signage, trash sorters, different bin designs, etc.)
- Comes up with notifications for facilities management
Leaderboards
- Incentivises zero waste to users
- Helps users develop zero waste habits
Waste Guidance
- Guides that help users locate nearest appropriate bins that are not full
- Educates users on zero waste
